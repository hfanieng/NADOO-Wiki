## 1. Python-Installation und virtuelle Umgebung

Bevor wir mit dem Coding beginnen, installieren wir Python 3.11 mithilfe von uv und erstellen eine virtuelle Umgebung.

### Python 3.11 Installation mit uv

1. Öffnen Sie das Terminal in Visual Studio Code (View > Terminal oder ``Ctrl+` ``).

2. Installieren Sie uv:

   Für macOS und Linux:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

   Für Windows (in PowerShell):
   ```powershell
   irm https://astral.sh/uv/install.ps1 | iex
   ```

3. Installieren Sie Python 3.11:
   ```bash
   uv python install 3.11
   ```

4. Überprüfen Sie die Installation:
   ```bash
   uv python --version
   ```

### Erstellen einer virtuellen Umgebung

1. Erstellen Sie eine virtuelle Umgebung:
   ```bash
   uv venv -p 3.11
   ```

2. Aktivieren Sie die virtuelle Umgebung:

   Für macOS und Linux:
   ```bash
   source .venv/bin/activate
   ```

   Für Windows:
   ```bash
   .\.venv\Scripts\activate
   ```

## 2. Python Grundkonzepte

Hier ist ein Beispielskript, das grundlegende Python-Konzepte demonstriert:

```python
# Variablen und Datentypen
zahl = 10  # Integer (Ganzzahl)
kommazahl = 3.14  # Float (Gleitkommazahl)
text = "Hallo, Welt!"  # String (Zeichenkette)
liste = [1, 2, 3, 4, 5]  # Liste (veränderbare Sequenz von Elementen)

# Funktionen
def addiere(a, b):
    return a + b

# Kontrollstrukturen
# If-Anweisung
if zahl == 10:
    print("Die Zahl ist 10")
else:
    print("Die Zahl ist nicht 10")

# For-Schleife
for i in range(5):
    print(f"Schleifendurchlauf: {i}")

# While-Schleife
counter = 0
while counter < 3:
    print(f"While-Schleife: Zähler = {counter}")
    counter += 1

# Klassen und Objekte
class Auto:
    def __init__(self, marke, modell):
        self.marke = marke
        self.modell = modell
    
    def fahren(self):
        print(f"{self.marke} {self.modell} fährt.")

# Fehlerbehandlung
try:
    resultat = 10 / 0
except ZeroDivisionError:
    print("Division durch Null ist nicht erlaubt")
finally:
    print("Dies wird immer ausgeführt")

# Hauptprogramm
if __name__ == "__main__":
    ergebnis = addiere(5, 3)
    print(f"5 + 3 = {ergebnis}")

    mein_auto = Auto("VW", "Golf")
    mein_auto.fahren()

print("Programm beendet")
```

## 3. Erklärung der Konzepte

1. **Variablen und Datentypen**: Variablen sind Behälter für Daten. Python unterstützt verschiedene Datentypen wie Integers, Floats, Strings und Listen.

2. **Funktionen**: Funktionen sind wiederverwendbare Codeblöcke. Sie werden mit `def` definiert und können Parameter akzeptieren und Werte zurückgeben.

3. **Kontrollstrukturen**:
   - If-Anweisungen ermöglichen bedingte Ausführung von Code.
   - For-Schleifen werden für Iterationen über Sequenzen verwendet.
   - While-Schleifen wiederholen Code, solange eine Bedingung wahr ist.

4. **Klassen und Objekte**: Klassen sind Blaupausen für Objekte. Sie können Attribute (Daten) und Methoden (Funktionen) enthalten.

5. **Fehlerbehandlung**: Try-Except-Blöcke werden verwendet, um mit möglichen Fehlern umzugehen und das Programm kontrolliert weiterlaufen zu lassen.

6. **Hauptprogramm**: Der `if __name__ == "__main__":` Block wird nur ausgeführt, wenn das Skript direkt gestartet wird, nicht wenn es als Modul importiert wird.

Diese Grundlagen bilden das Fundament für das Verständnis und die Entwicklung von Python-Programmen. Im nächsten Abschnitt werden wir uns mit dem Debugging dieser Konzepte befassen.
