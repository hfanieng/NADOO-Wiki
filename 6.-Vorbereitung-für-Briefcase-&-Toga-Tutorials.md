Bevor wir mit den Briefcase und Toga Tutorials beginnen, installieren wir Python 3.11 mithilfe von uv und erstellen eine virtuelle Umgebung. Python 3.11 ist die letzte garantiert kompatible Version für alle folgenden Tutorials.

## Python 3.11 Installation mit uv

1. Öffnen Sie das Terminal in Visual Studio Code (View > Terminal oder ``Ctrl+` ``).

2. Installieren Sie uv:

   Für macOS und Linux:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

   Für Windows (in PowerShell):
   ```powershell
   irm https://astral.sh/uv/install.ps1 | iex
   ```

3. Installieren Sie Python 3.11:
   ```bash
   uv python install 3.11
   ```

4. Überprüfen Sie die Installation:
   ```bash
   uv python --version
   ```

## Erstellen einer virtuellen Umgebung

Eine virtuelle Umgebung ist ein isolierter Bereich für projektspezifische Python-Pakete und Abhängigkeiten. Sie bietet mehrere Vorteile:

- Isolierung von Projekten
- Versionskontrolle für Bibliotheken
- Einfache Reproduzierbarkeit
- Saubere Entwicklungsumgebung
- Vereinfachtes Teilen von Projekten

### Installation der virtuellen Umgebung

1. Erstellen Sie eine virtuelle Umgebung:
   ```bash
   uv venv -p 3.11
   ```

2. Aktivieren Sie die virtuelle Umgebung:

   Für macOS und Linux:
   ```bash
   source .venv/bin/activate
   ```

   Für Windows:
   ```bash
   .\.venv\Scripts\activate
   ```

**Briefcase** ist ein Tool zur Erstellung plattformübergreifender Anwendungen in Python. Es ermöglicht das Verpacken und Verteilen von Python-Projekten als native Anwendungen für verschiedene Betriebssysteme.
